INTRODUCCION HTML

Este es un curso que pretende introducir al lector en el mundo del HTML. No se requieren conocimientos previos, se empezará desde cero. No hay que saber nada de programación para utilizar HTML, basta aprender un par de cosas sencillas para empezar a crear páginas.

Es MUY FACIL aprender HTML. Tan sólo necesitas un editor de texto, un curso de html y un navegador web (como netscape o internet explorer), ni siquiera necesitas estar conectado a Internet.
